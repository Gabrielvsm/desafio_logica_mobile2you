This app is a resolution of a challenge proposed by the company Mobile2you. In this challenge we were given an expression that would contain 'sands' ( . ) and 'diamonds' ( <> ). We were supposed to clean the _sand_ off the expression and then count how many _diamonds_ were extracted from it.

The expression given is '<<.<<..>><>><.>.>.<<.>.<.>>>><>><>>'. Where were found 7 diamonds after cleaning it, as shown when executing the script:

```bash
> ruby app.rb
After cleaning 7 diamonds were found
```
